 KiDS-GGL References

Acknowledegements
=================

If you use ``kids_ggl`` in your research, please cite |ref:dvornik17|_. You may optionally also cite |ref:brouwer16|_ and 
|ref:sifon15|_, for which early versions of ``kids_ggl`` were developed.


External resources
------------------

The development of ``kids_ggl`` has at many stages taken advantage of software developed as part of the `Jupyter project 
<https://jupyter.org/>`_.

In addition, ``kids_ggl`` uses the following Python packages, which should also be cited at appropriate places:

Standard Python packages
************************

* ``IPython`` (`<https://ipython.org/>`_)
* ``matplotlib`` (`<https://matplotlib.org/>`_; `Hunter 2007 <https://ieeexplore.ieee.org/document/4160265>`_)
* ``numpy`` (`<https://www.numpy.org/>`_)
* ``scipy`` (`<https://www.scipy.org/>`_)

Other packages
***************

* ``astropy``: Astronomical utility package (`Astropy Collaboration 2013 
  <https://ui.adsabs.harvard.edu/abs/2013A&A...558A..33A/abstract>`_, `2018 
  <https://ui.adsabs.harvard.edu/abs/2018AJ....156..123A/abstract>`_/`website <https://www.astropy.org/index.html>`_)
* ``CAMB``: Transfer function solver (`Lewis et al. 2000 <https://ui.adsabs.harvard.edu/abs/2000ApJ...538..473L/abstract>`_/`website <https://camb.info/readme.html>`_)
* ``emcee``: Halo model MCMC sampling (`Foreman-Mackey et al. 2013 
  <https://ui.adsabs.harvard.edu/abs/2013PASP..125..306F/abstract>`_/`website <https://emcee.readthedocs.io/en/latest/>`_), which 
  implements the Affine invariant Ensemble sampler of   `Goodman & Weare (2010) <https://msp.org/camcos/2010/5-1/p04.xhtml>`_.
* ``hmf``: Halo mass function (`Murray et al. 2013 
  <https://ui.adsabs.harvard.edu/abs/2013A%26C.....3...23M/abstract>`_/`website <https://hmf.readthedocs.io/en/latest/>`_)
