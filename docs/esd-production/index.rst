=========================
 KiDS-GGL ESD Production
=========================

The first module of ``kids_ggl`` allows the user to easily produce galaxy-galaxy lensing measurements around a pre-defined set of 
lens galaxies. Summaries of the process of calculating the galaxy-galaxy lensing signal can be found in |ref:viola15|_ and 
|ref:dvornik17|_.
